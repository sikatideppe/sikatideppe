## 📌 Projet phare – Base de données avancée (SQL Server)

### 🚦 Analyse des données de trafic routier – **Ontario 511**

🔗 **Lien du projet GitHub :**
[https://github.com/sikatideppe/Projet_Base_de_donn-e_Avanc-_UA3](https://github.com/sikatideppe/Projet_Base_de_donn-e_Avanc-_UA3)

**Contexte :**
Ce projet repose sur l’exploitation des données du système **Ontario 511**, qui fournit des informations en temps réel et historiques sur le trafic routier.

**Objectifs :**

* Concevoir et implémenter une **base de données relationnelle avancée**
* Exploiter les données de trafic pour répondre à des questions analytiques concrètes
* Mettre en pratique les concepts avancés de **SQL Server**

**Compétences mises en œuvre :**

* Modélisation de bases de données (schéma relationnel)
* Requêtes SQL avancées (jointures complexes, sous-requêtes)
* Gestion et structuration de données volumineuses
* Analyse de données orientée aide à la décision

**Valeur ajoutée :**
Ce projet démontre ma capacité à travailler sur des **données réelles**, à structurer efficacement l’information et à produire des analyses exploitables dans un contexte professionnel.
